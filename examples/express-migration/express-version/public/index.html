TaskAPI - Express Version

A reference REST API demonstrating Express patterns for bunway migration testing.

Available Endpoints

GET / - API info
GET /api/health - Health check
POST /api/auth/login - Login (username, password)
POST /api/auth/logout - Logout
GET /api/auth/me - Get current user (requires auth)
GET /api/tasks - List tasks (requires auth)
POST /api/tasks - Create task (requires auth)
GET /api/tasks/:id - Get task (requires auth)
PUT /api/tasks/:id - Update task (requires auth)
DELETE /api/tasks/:id - Delete task (requires auth)
POST /api/upload/avatar - Upload avatar image (requires auth)

Test Users

Example Request

curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "password123"}' \
  -c cookies.txt

curl -X GET http://localhost:3000/api/tasks \
  -b cookies.txt
